# 🔮 AI 塔羅占卜 Pro v2.0

全功能塔羅占卜網頁 APP，支援 AI 智慧解讀、多種牌陣、正逆位、歷史記錄、統計分析！

## ✨ 功能清單

### 🃏 牌陣系統
| 牌陣 | 說明 | 張數 |
|------|------|------|
| 🎴 單牌占卜 | 快速得到指引 | 1 |
| ❓ 是非題 | 回答是/否問題 | 1 |
| ⏳ 時間之流 | 過去→現在→未來 | 3 |
| 💕 愛情牌陣 | 感情關係解析 | 4 |
| 💼 事業牌陣 | 職場工作指引 | 4 |
| ☀️ 每日運勢 | 今日整體運勢 | 3 |

### 🤖 AI 智慧解讀
- 輸入問題，AI 根據你的問題客製化解讀
- 支援 Groq API（Llama 3.3 70B）
- 分析牌組關聯意義
- 提供具體建議和行動方向

### 🔄 正逆位系統
- 可開關正逆位功能
- 22 張牌都有正位/逆位牌義
- 逆位牌面旋轉 180 度顯示

### 🗣️ 語音播報
- Web Speech API（免費）
- Gemini TTS（需 API Key）
- 朗讀牌義和建議

### 📜 歷史記錄
- 自動儲存每次占卜
- 最多保留 100 筆記錄
- 可回顧查看歷史

### 📊 統計分析
- 總占卜次數
- 本週占卜次數
- 平均運勢分數
- 連續占卜天數
- 最常抽到的牌 TOP 5

### 📚 牌義教學
- 22 張大阿爾克那完整牌義
- 正位/逆位詳細說明
- 關鍵詞和建議

### 🎨 視覺效果
- 冷光科技開機動畫
- 翻牌 3D 動畫
- 洗牌/翻牌音效
- iPhone 風格 UI

### 📱 LINE 推送
- Flex Message 精美卡片
- 圖文並茂呈現結果
- 包含 AI 解讀內容

## 🚀 快速開始

### 1. 開啟網頁
直接用瀏覽器開啟 `index.html`

### 2. 設定 API（選填）
- **Groq API Key**：啟用 AI 解讀
- **GAS URL**：啟用 LINE 推送

### 3. 開始占卜！
1. 選擇牌陣
2. 輸入問題（選填）
3. 選擇翻牌模式
4. 按「開始占卜」

---

## ⚙️ API 設定

### Groq API（AI 解讀）
1. 前往 [Groq Console](https://console.groq.com/)
2. 註冊帳號
3. 建立 API Key
4. 貼到設定頁

### LINE 通知
1. 建立 LINE Bot（參考 GAS 說明）
2. 部署 `tarot-gas.js` 到 Google Apps Script
3. 複製部署網址到設定頁
