# 🔮 塔羅占卜 & 八字命盤

iPhone 風格的命理占卜網頁 APP，支援 LINE Flex Message 圖文推送！

## ✨ 功能特色

### 🃏 塔羅占卜
- 22 張大阿爾克那完整牌義
- 每日運勢、愛情、事業、財運
- 翻牌動畫效果
- 詳細牌義解讀與建議

### 📅 八字命盤
- 四柱八字計算
- 五行分析圖表
- 日主特質解讀
- 開運顏色與方位建議

### 📱 LINE 圖文推送
- **Flex Message** 精美卡片設計
- 圖片 + 文字 + 排版完整呈現
- 塔羅牌面圖片
- 五行元素圖示

## 🚀 快速開始

### 線上使用
直接用瀏覽器開啟 `index.html`

### 部署到 GitHub Pages
```bash
git init
git add .
git commit -m "🔮 塔羅占卜 & 八字命盤"
git remote add origin https://github.com/YOUR_USERNAME/tarot-bazi.git
git push -u origin main
```

## ⚙️ 設定 LINE 通知

### Step 1：建立 LINE Bot
1. 前往 [LINE Developers](https://developers.line.biz/)
2. 建立 Provider 和 Channel（Messaging API）
3. 取得 **Channel Access Token**
4. 取得你的 **User ID**

### Step 2：部署 GAS
1. 開啟 [Google Apps Script](https://script.google.com/)
2. 新增專案，貼上 `tarot-bazi-gas.js` 內容
3. 修改 LINE 設定：
```javascript
const LINE_CHANNEL_TOKEN = '你的 Channel Access Token';
const LINE_USER_ID = '你的 User ID';
```
4. 部署 → 新增部署 → 網頁應用程式
5. 複製部署網址

### Step 3：設定網頁 APP
1. 開啟網頁 → 點擊 ⚙️ 設定
2. 貼上 GAS URL
3. 完成！
